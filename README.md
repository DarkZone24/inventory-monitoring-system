# SchoolAsset: Inventory Monitoring System

This project is a High School Property Management System built with React, Vite, and Express.

## Getting Started

1. **Setup Backend**:
   - Go to `server/` folder.
   - Run `npm install`.
   - Configure `.env` based on `.env.example`.
2. **Setup Frontend**:
   - Return to root folder.
   - Run `npm install`.
3. **Run Locally**:
   - Run `npm run dev-all` to start both frontend and backend.
